# Todo List App - Fullstack

פרויקט ניהול משימות המשלב צד לקוח (React) וצד שרת (ASP.NET Core).

## תכונות הפרויקט
* **הרשמה והתחברות**: מערכת אבטחה המבוססת על JWT.
* **ניהול משימות**: יצירה, עריכה, מחיקה וסימון משימות שבוצעו.
* **מסד נתונים**: עבודה עם MySQL בשיטת Entity Framework.

## איך להריץ את הפרויקט?

### צד שרת (Backend)
1. יש לוודא שמותקן .NET SDK.
2. עדכון מחרוזת החיבור ב-`appsettings.json`.
3. הרצת הפקודה `dotnet run`.

### צד לקוח (Frontend)
1. כניסה לתיקיית הפרוייקט.
2. הרצת `npm install` להתקנת הספריות.
3. הרצת `npm start`.

## טכנולוגיות
* React
* ASP.NET Core Web API
* MySQL / Entity Framework
* Axios & JWT
